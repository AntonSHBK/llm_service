## 1. JSON Mode

### –û–ø–∏—Å–∞–Ω–∏–µ

JSON Mode ‚Äî –±–∞–∑–æ–≤—ã–π —Ä–µ–∂–∏–º —Ä–∞–±–æ—Ç—ã, –ø—Ä–∏ –∫–æ—Ç–æ—Ä–æ–º –º–æ–¥–µ–ª—å –≥–∞—Ä–∞–Ω—Ç–∏—Ä—É–µ—Ç, —á—Ç–æ —Ä–µ–∑—É–ª—å—Ç–∞—Ç –±—É–¥–µ—Ç –≤–∞–ª–∏–¥–Ω—ã–º JSON-–æ–±—ä–µ–∫—Ç–æ–º.

### –û—Å–æ–±–µ–Ω–Ω–æ—Å—Ç–∏

* –û–±–µ—Å–ø–µ—á–∏–≤–∞–µ—Ç –∫–æ—Ä—Ä–µ–∫—Ç–Ω—ã–π —Å–∏–Ω—Ç–∞–∫—Å–∏—Å JSON.

### –ü—Ä–∏–º–µ—Ä (Python)

```python

    model="gpt-3.5-turbo-0125",

        {"role": "user", "content": "Who won the world series in 2020? Return in format {winner: ...}"}
    ],
    text={"format": {"type": "json_object"}}
)

print(response.output_text)
# –ü—Ä–∏–º–µ—Ä —Ä–µ–∑—É–ª—å—Ç–∞—Ç–∞: {"winner": "Los Angeles Dodgers"}
```
